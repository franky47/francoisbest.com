# Open Source

This page lists some of the open source packages and repositories I have published or contribute to.

Most of them are published under my company [47ng](https://github.com/47ng).

## Frontend

My tech stack of choice is [Next.js](https://nextjs.org) and [Chakra UI](https://chakra-ui.com).

Next.js makes it super easy to build static or dynamic webapps, depending on
what is needed.

Chakra UI offers accessibility out of the box with great default styles, that are
easy to override thanks to the styled-system approach.

## Backend

I like to use [Fastify](https://fastify.io) to build backend services in Node.js.

What I like about it:

- More opinionated and structured than Express.js
- Damn fast
- Easy to write plugins
- Good defaults out of the box
